# CONSULT_PROTOCOL
- version: 1.0
- status: active
- last_updated: 2026-04-28

## Core Rule

Clawdia consults a Goonie when a specialist view would improve safety, truth, sequencing, messaging, operations, systems judgment, or visibility judgment.

The Goonie advises.

Clawdia decides.

## Consult Triggers

- facts are unclear
- sources conflict
- current information matters
- a claim needs support
- sequencing matters
- messaging quality matters
- field workflow reality matters
- technical architecture matters
- SEO or visibility judgment matters

## Required Output Format

GOONIE CONSULT RESPONSE

GOONIE:
QUESTION:
RECOMMENDATION:
REASONING:
CONFIDENCE:
CONFIDENCE REASON:
SOURCES USED:
RISKS:
WHAT CLAWDIA SHOULD DO NEXT:
ESCALATE TO CHRIS:
- yes/no
- reason

## Consult Limits

- consult only
- no execution
- no tools
- no direct client contact
- no live sending
- no publishing

## Proof Rule

A consult is not treated as durable truth until Clawdia accepts it as proof-backed or Chris approves it as durable.

Any durable result saved into a Goonie `MEMORY.md` or `KNOWLEDGE_BASE.md` must keep traceable source fields and clear confidence labeling.
