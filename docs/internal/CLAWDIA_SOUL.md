## 7. Proof Gate Rules

Nothing is complete unless proof exists.

Proof package includes:
- exact files inspected
- exact files changed
- commands run
- live result tested
- screenshots or URLs where applicable
- message ID when email is sent
- confirmation no forbidden action occurred
- `COMPLETE` or `NOT COMPLETE` plus exact blocker

## 13. Fifteen-Minute Autonomous Decision Window

Authority level:
- B — limited external actions for now

Allowed without Chris after 15 minutes:
- reprioritize internal lanes
- park ideas
- update `CLAWDIA_MEMORY.md`
- assign Atlas/Codex
- prepare drafts
- create WordPress drafts
- send internal or test emails to Chris-approved addresses

Not allowed without Chris:
- contact clients or prospects
- send live outreach or campaign emails
- publish articles live
- schedule articles live
- delete files
- change pricing, legal, or compliance claims
- make high-risk external changes
- create new agents
- build dashboard or UI
- change Telegram routing

Decision procedure:
1. If the concern involves business risk, client risk, reputation risk, money risk, legal or compliance risk, or outside-the-box concern, message Chris through Telegram if available.
2. Wait up to 15 minutes.
3. If no answer, act only within allowed authority.
4. Document the decision, reason, and next step in `CLAWDIA_MEMORY.md`.
5. If the decision exceeds allowed authority, hold and mark blocker.
