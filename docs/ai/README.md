# AI Workflow System

This folder contains the agent definitions, prompt templates, and
workflow instructions for the Aquatrace AI agent system.

## What Is In This Folder

| File or Folder | Purpose |
|---|---|
| agent-schema.md | Master template every agent is built from |
| agent-architect.md | Agent Architect definition |
| agent-intake.md | Intake Agent definition |
| agent-planner.md | Planner Agent definition |
| agent-builder.md | Builder Agent definition |
| agent-qa.md | QA / Review Agent definition |
| prompts/ | Reusable Claude and Codex prompt templates |
| workflow/ | Daily use guide, ticket template, plan template, approval rules |

## How This System Works

Claude handles planning, design, and review.
Codex handles implementation and file changes.
You are the human operator. Nothing is approved without you.

Full daily use guide: docs/ai/workflow/how-to-use-this-system.md
