# Dive Factor Underwater Services - Pricing Framework

## Purpose

This file defines the internal pricing framework for milestone-one planning.

It is not a public rate sheet.

## Starting anchor

- internal starting range: $295+

This anchor should represent a minimum-scope local visual inspection with standard turnaround and uncomplicated access.

## What the anchor is not

- not a public promise
- not a flat fee for every scope
- not valid for travel-heavy, urgent, or multi-target jobs

## Quote drivers

Use these variables in every future quote path:

1. target type
2. target count
3. boat, dock, lift, or property size
4. travel zone
5. access difficulty
6. water conditions and likely visibility
7. documentation depth
8. urgency or rush turnaround
9. partner or reporting complexity

## Internal quote bands

### Band 1 - Minimum scope

- one target
- local travel
- standard photo/video capture
- concise written summary

### Band 2 - Expanded documentation

- larger target or more media capture
- fuller written report
- better fit for pre-purchase or caution-driven requests

### Band 3 - Extended or priority scope

- multiple targets
- travel-heavy scope
- faster turnaround
- best fit for marina, HOA, or larger-property use cases

## Pricing rules

- keep travel and urgency as explicit line items later
- separate report-depth upgrades from base scope later
- do not use certification, engineering, or insurance-acceptance language to justify price
- do not release a public price sheet until Chris approves packaging
