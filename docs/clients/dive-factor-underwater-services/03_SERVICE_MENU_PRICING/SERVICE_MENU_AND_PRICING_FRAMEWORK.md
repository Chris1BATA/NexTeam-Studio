# Dive Factor Underwater Services - Service Menu and Pricing Framework

## Launch logic

Start with inspection and documentation services first.

Why:

- easier to explain
- easier to package
- lower claim risk than repair or certification language
- strong fit for boats, docks, lifts, marinas, and lake homes

These should not be sold or publicly claimed until operations, insurance, and compliance review are complete:

1. Coastal hull cleaning and anode or zinc service
2. Marina recurring maintenance contracts
3. Rope or line removal in complex entanglement conditions
4. Heavy debris removal
5. Structural certification language

## Quote framework

Use a quote path instead of a public menu price at this stage.

### Base-scope anchor

- internal starting anchor: $295+
- intended to represent a minimum-scope local visual inspection with standard turnaround

### Main quote drivers

1. target type: boat, dock, lift, swim area, shoreline zone, or mixed property scope
2. target count: single target vs. bundled scope
3. vessel or structure size
4. travel zone and launch logistics
5. water conditions and expected visibility
6. access difficulty or permission complexity
7. documentation depth: basic, standard, or report-heavy
8. turnaround speed: standard vs. rush
9. referral coordination or follow-up review needs

### Internal pricing bands

- band 1: local single-target visual check
- band 2: larger target or fuller documentation report
- band 3: multi-target, travel-heavy, or priority-turnaround scope

Do not turn these bands into public price promises yet.

## Working package structure

### Package A - Basic visual check

- single target
- short photo/video capture
- concise written summary
- best fit: routine homeowner or boat-owner peace-of-mind check

### Package B - Documentation report

- more complete photo/video capture
- fuller written report
- clearer issue tagging
- better fit for pre-purchase or insurance-adjacent documentation
- best fit: higher-value boats, more cautious buyers, or second-home owners

### Package C - Priority or extended scope

- multi-target inspection
- longer travel or urgent turnaround
- best for marina, HOA, or larger property use cases
- may become the bridge to B2B partner plans after operating review
