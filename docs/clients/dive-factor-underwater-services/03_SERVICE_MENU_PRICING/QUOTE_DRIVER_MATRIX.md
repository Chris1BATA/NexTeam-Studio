# Dive Factor Underwater Services - Quote Driver Matrix

| Driver | Low complexity | Medium complexity | High complexity |
| --- | --- | --- | --- |
| Target count | single | two related targets | multi-target property or fleet |
| Asset size | small | medium | large or commercial |
| Travel | local | extended local | out-of-zone |
| Access | straightforward | moderate setup | difficult or controlled access |
| Visibility | good | limited | poor |
| Documentation depth | concise summary | fuller report | report-heavy package |
| Urgency | standard | priority | rush / after-hours |
| Stakeholder complexity | single owner | owner plus manager | multi-party or partner reporting |

## Use

This matrix should guide internal quoting consistency before any calculator is built.
