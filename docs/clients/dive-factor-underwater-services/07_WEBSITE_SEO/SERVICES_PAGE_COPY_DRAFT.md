# Services Page Copy Draft

Internal draft only. Not approved for publishing.

## Page purpose

Explain the phase-one service set in simple, safe language.

## Intro

Dive Factor is being planned around underwater visual checks and documentation for common lakefront concerns. The phase-one service set is intentionally narrow so the offer stays clear, useful, and safely worded.

## Featured service

### Below-Waterline Inspection

A practical underwater check designed to help owners see the condition of key areas below the surface. This service is intended for boats, docks, lifts, swim areas, and selected waterfront concerns that fit the phase-one scope.

## Supporting services

### Dock and Lift Visual Check

Focused visual review of visible underwater dock or lift areas, with photo/video documentation when practical and written notes on what was observed.

### Prop and Running Gear Visual Check

For owners who want a visual look at the propeller or accessible running-gear area when a below-waterline concern is suspected.

### Swim-Area Scan

A visual underwater scan of a defined swim area to help identify obvious visible concerns or hazards.

## Secondary / situational service

### Lost Item Recovery

This remains a secondary, situational service idea for screened conditions. It should not be presented as a broad recovery or salvage promise.

## What customers receive

- underwater photo/video documentation when practical
- a written summary of visual observations
- notes on visible limitations or next-step referral needs

## Scope note

This draft service page is intentionally visual and documentation-led. It does not present structural certification, emergency-response guarantees, or final public pricing.
