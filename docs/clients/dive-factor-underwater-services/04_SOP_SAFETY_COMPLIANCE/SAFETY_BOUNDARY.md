# Dive Factor Underwater Services - Safety Boundary

## Purpose

This file defines the planning-time safety and public-claim boundary for Dive Factor.

It is not legal advice and does not authorize public claims by itself.

## Hard boundary

Do not publicly claim any of the following unless Chris verifies proof:

- OSHA compliant
- commercial diving contractor
- certified structural inspection
- engineering-grade inspection
- code-certified inspection
- insurance-approved contractor
- emergency-response readiness

## Safer planning language

These phrases are lower-risk if kept precise:

- underwater photo and video documentation
- visual check
- visual condition notes
- below-waterline inspection
- swim-area hazard scan
- written summary
- repair or referral recommendation

## Service boundary

Allowed to package internally now:

- inspection and documentation concepts
- visual checks
- swim-area scans
- storm-damage documentation
- pre-purchase documentation

Not allowed to present as approved public capabilities yet:

- structural certification
- regulatory compliance claims
- maintenance-heavy coastal work
- emergency or salvage positioning

## Operating rule

If a service requires legal, insurance, OSHA, environmental, or operational review, mark it `hold for review` and keep it out of public-facing materials.
