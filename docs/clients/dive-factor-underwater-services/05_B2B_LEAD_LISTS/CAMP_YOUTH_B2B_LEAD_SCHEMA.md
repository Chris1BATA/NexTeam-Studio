# Camp & Youth B2B Lead Schema

Internal-only lead qualification schema. Not approved for outreach use.

Last updated: 2026-06-15

## Purpose

Standardize how Dive Factor evaluates camps, schools, youth groups, and aquatic operators for this future lane without mixing them into the marina or underwater services pipeline.

## Lead categories

- Christian summer camps
- YMCA camps
- private schools
- church youth groups
- lake camps
- retreat centers
- homeschool co-ops
- scout and outdoor groups
- parks and recreation departments
- swim teams
- lifeguard teams
- waterfront staff programs

## Core lead fields

### Identity

- organization name
- lead category
- website
- city
- state
- camp season dates
- operating model: day camp / resident camp / school / church / civic / private operator

### Decision-maker data

- primary contact name
- title
- email
- phone
- secondary contact
- decision stage

### Aquatic environment data

- facility type: pool / lake / waterfront / mixed
- pool available: yes / no / unknown
- pool depth profile known: yes / no
- standing-depth area available: yes / no / unknown
- lifeguard coverage available: yes / no / unknown
- waterfront director or aquatics lead present: yes / no / unknown

### Program fit data

- interest lane: youth scuba / snorkeling confidence / staff safety / emergency readiness / bundle
- target age band
- estimated participant count
- staff count needing training
- preferred delivery window
- indoor backup available

### Risk and compliance data

- parent permission workflow exists: yes / no / unknown
- medical form workflow exists: yes / no / unknown
- vendor packet required: yes / no / unknown
- COI required: yes / no / unknown
- additional insured requested: yes / no / unknown
- abuse/molestation coverage requested: yes / no / unknown
- ACA-accredited: yes / no / unknown

### Commercial fit data

- budget posture: premium / moderate / cost-sensitive / unknown
- travel burden: low / medium / high
- repeat potential: low / medium / high
- cross-sell potential: low / medium / high
- operational complexity: low / medium / high

## Qualification tags

- teen-friendly pool fit
- likely staff-training-first
- likely scuba-fit later
- likely snorkeling-only
- requires broker review
- requires contract review
- requires high staffing
- not a fit

## Disqualification flags

- no controlled aquatic environment
- no camp-side supervision support
- no paperwork path for minors
- unrealistic group size for staffing model
- expects entertainment pricing for high-liability activity
- wants public/open drop-in style programming

## Internal scoring suggestion

### Strong early-fit leads

- teen population
- pool access
- organized waterfront team
- staff training need
- premium safety culture
- repeat seasonal potential

### Weak early-fit leads

- no pool
- unclear supervision
- large mixed-age demand
- price-shopping behavior
- vague expectations

## Data collection note

Do not collect or use this schema for outreach yet.

Use it only to prepare future research lists and internal qualification logic after Chris approves the lane.
