# AQUATRACE CASE STUDY LIBRARY

## Current Status

- Case-study structure is prepared.
- No customer-private case-study details should be added without approval.

## Approved Case-Study Pattern

- leak type or symptom
- site type: residential or commercial
- diagnostic methods used
- what was documented
- what the customer or operator needed next
- no private customer identity
- no exact street address

## Candidate Case-Study Themes

- pool leak that seemed to stop but reopened later
- skimmer leak confirmed through dye testing
- plumbing leak isolated through pressure testing
- light niche leak identified during underwater inspection
- commercial pool documentation flow for operator decision-making
