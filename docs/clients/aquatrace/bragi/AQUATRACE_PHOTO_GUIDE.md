# AQUATRACE PHOTO GUIDE

## Core Rule

Each Bragi photo recommendation must describe one clear photo type only.

## Required Package Coverage

- 1 featured image recommendation
- 3 to 5 supporting photo recommendations
- each photo needs placement, filename, title, alt text, caption, and description

## Good Photo Types

- Technician pressure testing pool plumbing at the equipment pad
- Bucket test setup beside a residential pool
- Technician dye testing around a skimmer throat
- Underwater inspection at a pool light niche
- Equipment pad overview showing leak-detection setup
- Commercial pool technician documenting suspected leak points

## Bad Photo Types

- Mixed scene descriptions that combine too many subjects
- Photos that do not match the claimed leak type
- Photos with unclear content where metadata would require guessing

## Metadata Rule

- Final metadata must match the actual uploaded photo.
- If the actual image differs from the recommendation, Bragi must rewrite filename, alt text, caption, and description.
- If content is unclear, mark `needs human review`.
