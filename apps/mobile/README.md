# Mobile App

This folder is reserved for the future M11 NexTeam mobile app. No active runtime ships from here yet.

It will connect to `apps/server` APIs and shared contracts from `@nexteam/core` when the mobile wave opens. Until then, do not add client-specific behavior here unless the M11 module is explicitly active.

When something breaks, first confirm whether M11 has opened. If it has not, this folder should remain a placeholder and should not affect current builds.
