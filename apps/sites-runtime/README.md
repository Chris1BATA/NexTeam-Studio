# Sites Runtime

This folder is reserved for the future M8 tenant website runtime and static-site generation work. No active site publishing should run from here before that wave opens.

It will eventually consume tenant data, content artifacts, and approved site publish items from the server-side modules. WordPress or site-publishing credentials must never be added here before M5/M8 scope allows it.

When something breaks, first confirm whether the sites wave is active. If it is not active, this folder should remain inert and should not affect current builds.
